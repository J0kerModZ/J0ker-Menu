# J0ker-Menu
Source Code For J0kers Gorilla Tag Menu

# Installation
 - Download the menu [here](https://github.com/J0kerModZ/J0ker-Menu/releases/latest)
- Drag the file ``J0kerMenu GTAG.dll`` to your plugins folder
- Start Gorilla Tag
